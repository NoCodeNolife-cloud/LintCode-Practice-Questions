给定一个正整数数组nums。
输出连续的子数组个数，每个子数组中元素的乘积小于k。

样例
样例 1:
	输入:  nums = [10, 5, 2, 6], k = 100
	输出:  8
	
	解释:
	这8个子段是: [10], [5], [2], [6], [10, 5], [5, 2], [2, 6], [5, 2, 6].
	[10, 5, 2] 没有严格小于100所以不算。

	
解释 2:
	输入: nums = [5,10,2], k = 10
	输出:  2
	
	解释:
	只有 [5] 和 [2].
注意事项
0 < nums.length <= 50000.
0 < nums[i] < 1000.
0 <= k < 10^6.
