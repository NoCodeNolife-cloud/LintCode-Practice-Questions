写出一个函数 `anagram(s, t)` 判断两个字符串是否可以通过改变字母的顺序变成一样的字符串.

### 样例

**样例 1:**

```
输入: s = "ab", t = "ab"
输出: true
```

**样例 2:**

```
输入:  s = "abcd", t = "dcba"
输出: true
```

**样例 3:**

```
输入:  s = "ac", t = "ab"
输出: false
```

### 挑战

O(n) 的时间复杂度, O(1) 的额外空间

### 说明

什么是 **Anagram**?

-    在更改字符顺序后两个字符串可以相同

```cpp
class Solution {
public:
    /**
     * @param s: The first string
     * @param t: The second string
     * @return: true or false
     */
    bool anagram(string &s, string &t) {
        sort(s.begin(),s.end());
        sort(t.begin(),t.end());
        if(s==t){
            return true;
        }
        else{
            return false;
        }
    }
};
```
