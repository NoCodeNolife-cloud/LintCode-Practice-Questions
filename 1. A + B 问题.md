给出两个整数 a和 b , 求他们的和.

### 样例

**样例 1:**

```
输入:  a = 1, b = 2
输出: 3	
样例解释: 返回a+b的结果.
```

**样例 2:**

```
输入:  a = -1, b = 1
输出: 0	
样例解释: 返回a+b的结果.
```

### 挑战

显然你可以直接 return a + b,但是你是否可以挑战一下不这样做？(不使用++等算数运算符)

### 说明

a和b都是 `32位` 整数么？

-    是的

我可以使用位运算符么？

-    当然可以

### 注意事项

你不需要从输入流读入数据,只需要根据`aplusb`的两个参数a和b,计算他们的和并返回就行.

```cpp
class Solution {
public:
    /**
     * @param a: An integer
     * @param b: An integer
     * @return: The sum of a and b 
     */
    int aplusb(int a, int b) {
        return a+b;
    }
};
```
