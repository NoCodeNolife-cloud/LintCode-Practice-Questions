约翰想在他家后面的空地上建一个后花园，现在有两种砖，一种3 dm的高度，7 dm的高度。约翰想围成x dm的墙。如果约翰能做到，输出YES，否则输出NO。

样例
样例 1:

输入 : x = 10
输出 : "YES"
解释：
x = 3 + 7 : 即需要1匹3 dm高度的砖和1匹7 dm 高度的砖。
样例 2:

输入 : x = 5
输出 : "NO"
解释：
不能用高度为3 dm的砖和高度为7 dm的砖砌成 5 dm的墙。
样例 3:

输入 : x = 13
输出 : "YES"
解释：
x = 2 * 3 + 7 : 即需要2匹3 dm高度的砖和1匹7 dm 高度的砖。
注意事项
X是一个整数，取值范围为 [3, 1000]。
